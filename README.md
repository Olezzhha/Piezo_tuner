# Piezo_tuner
Многофункциональный гитарный тюнер на пьезоэлектрическом датчике

ааа как редачить

На данный момент программы реализуют распознавание гитарных частот шестиструнной акустической гитары с выводом разницы между эталонным и действительным значением, номером струны и советом (подтянуть/ослабить/идеально).
Использованы алгоритмы: FFT, HPS, параболическая интерполяция, медианный фильтр
Помимо этого, на физической установке подключен джек для создания трансакустической гитары.

В планах:

1 - протестировать и сравнить другие алгоритмы обработки, не переводя сигнал в частотную область (в питоне уже тестировалось)
2 - добавить настройку в любые дропы, ниже классического (Drop D, Drop E...). Можно сделать распознавание более высоких дропов с помощью каподастра, но пока не знаю куда это использовать помимо отладки гитары (для идеальной настройки проверяются открытые струны и звучание на 12м ладу, если гитара не строит, требуется подкрутка анкера, подпилка порожков и тд).
3 - распознавание мажорного/минорного звучания аккорда
4 - определение акордов
5 - пока не придумала
